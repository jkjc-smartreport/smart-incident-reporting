üì¶ REQUIREMENTS TO INSTALL (on any new computer)
------------------------------------------------
1. Install **Python 3.10+**
   üëâ Download: https://www.python.org/downloads/

2. Install **MySQL Community Server**
   üëâ Download: https://dev.mysql.com/downloads/mysql/

3. Install **MySQL Workbench** (optional but helpful)
   üëâ Download: https://dev.mysql.com/downloads/workbench/

4. Install **pip** (Python package manager)
   Usually installed with Python automatically.
   To check:
       pip --version

5. Clone or copy your project folder (e.g., smart_incident_reporting)
   Make sure it contains:
       - app.py
       - templates/
       - static/
       - config.py
       - requirements.txt (if available)

------------------------------------------------
üß∞ SETUP MYSQL DATABASE
------------------------------------------------
1. Open MySQL Workbench or terminal.
2. Create a database, for example:
       CREATE DATABASE incident_db;
3. Import your SQL dump if available (your tables and data).

4. Update your config.py with your new credentials:
       import mysql.connector

       def get_db_connection():
           return mysql.connector.connect(
               host="localhost",
               user="root",
               password="YOUR_PASSWORD",
               database="incident_db"
           )

üåç MAKE YOUR APP ONLINE (HTTPS USING NGROK)
------------------------------------------------
1. Download **ngrok** from:
   üëâ https://ngrok.com/download

2. Sign up at https://dashboard.ngrok.com/
   Copy your **AuthToken** from your dashboard.

3. In Command Prompt, run:
       ngrok config add-authtoken YOUR_AUTHTOKEN

4. Start your Flask app:
       python app.py

5. Open another terminal (while Flask is running) and run:
       ngrok http 5000

6. You‚Äôll get a link like this:
       https://randomname.ngrok-free.app

7. Share that link ‚Äî it‚Äôs your HTTPS public version of your app!
